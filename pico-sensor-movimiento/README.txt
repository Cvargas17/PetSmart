# Sensor de Movimiento con Láser

## Hardware
- Raspberry Pi Pico W
- Láser 5V
- Sensor TEMT6000

## Conexiones
- Sensor OUT → GP28 (pin 34)
- Sensor VCC → 3.3V
- Sensor GND → GND
- Láser + → VBUS (pin 40)
- Láser - → GND

## Configuración
Modificar en main.py:
- SSID y PASSWORD de tu WiFi
- API_URL con la IP del servidor

## Uso
Ejecutar main.py en Thonny
